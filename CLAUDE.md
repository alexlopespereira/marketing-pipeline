# CLAUDE.md — Projeto de demonstração da aula

> **Atenção didática:** este arquivo é o CLAUDE.md — **configuração do sistema**.
> Ele NÃO é um arquivo de fundação. Os arquivos de fundação estão em
> `./foundation/` e são criados pelo marketeiro, sobre audiência, voz, mercado
> e jornada. Esta distinção é o ponto do Bloco 2 da aula.

## Sobre este projeto

Projeto de demonstração para a aula "Pipeline de Marketing com Claude Code"
(MBA em Marketing). Empresa fictícia: **GiroLeve** — SaaS B2B brasileiro de
fluxo de caixa e conciliação financeira para pequenos e médios e-commerces.

## Estrutura

- `./foundation/` — os 4 arquivos de fundação. **Conhecimento**, nunca executado.
- `./skills/` — as skills. **Execução**, acionadas pelo usuário.
- `./material-bruto/` — insumo para a Demo 1.
- `./plano-b/` — versões coláveis, caso o terminal falhe.

## Regra permanente

Todo conteúdo gerado neste projeto é **fictício e didático**. Nunca apresente
os dados da GiroLeve como reais.
