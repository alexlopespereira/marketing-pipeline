# Prompts — Etapa 2: Hub (peça-pivô) + 5 Hooks

> A regra desta etapa: o Hub nasce de **insumo humano**, não de conhecimento geral do modelo.

---

## Prompt 2.1 — Destilar o insumo humano ANTES de escrever

```
Aja como a skill skills/skill-hook-hub.md. Faça primeiro APENAS o Passo 2:
leia insumos/referencia-humana-entrevista.md e extraia:
- a TESE em uma frase (a opinião que só esta marca defenderia);
- 3 a 5 PROVAS literais, com a origem de cada uma;
- 1 INIMIGO (a ideia comum que a tese contraria).
Não escreva o Hub ainda. Só me devolva a destilação para eu aprovar.
```

## Prompt 2.2 — Escrever o Hub (após aprovar a destilação)

```
Aprovada a destilação. Agora execute os Passos 1, 3 e 5 da skill-hook-hub.md.
Ângulo escolhido da Etapa 1: [colar o ângulo]
Formato do Hub: [artigo de fundo ~1.200 palavras OU roteiro de videocast 15 min]

Carregue foundation/creator-style-EXEMPLO.md e
foundation/audience-delight-profile-EXEMPLO.md e declare que carregou.
Toda seção precisa se apoiar em uma das PROVAS. Use só o vocabulário da coluna
"Ela diz". Teste cada parágrafo contra a seção COMPARAÇÃO do Creator Style.
```

## Prompt 2.3 — A Hook Outline: 5 gatilhos distintos

```
Agora o Passo 4: gere 5 hooks para este Hub, uma por gatilho, sem repetir tipo:
1) contrariante  2) história pessoal  3) dado chocante  4) curiosity gap  5) prova/autoridade
Para cada uma: escreva SÓ a primeira linha (os 3 primeiros segundos), que precisa
funcionar isolada, e marque de qual PROVA do Passo 2 ela nasce. Nada de "no cenário
atual" nem "você já se perguntou". Comece no meio da ação.
```

## Prompt 2.4 — Estresse: as 5 hooks são realmente diferentes?

```
As 5 hooks têm ESTRUTURAS diferentes ou são a mesma frase reordenada? Para cada par,
diga em uma linha o que muda mecanicamente. Reescreva qualquer hook que seja sinônimo
de outra. Depois recomende qual você abriria e por quê (uma frase).
```

---

### Saída esperada da Etapa 2 (checklist do aluno)
- [ ] Destilação do insumo humano (tese, provas com origem, inimigo).
- [ ] Hub com toda seção ancorada em uma prova; voz testada contra a COMPARAÇÃO.
- [ ] 5 hooks, 5 gatilhos distintos, cada uma amarrada a uma prova.
- [ ] Recomendação de abertura.
